
Feature: : Cadastro de Usuário
  Como um usuário do sistema
  Quero cadastrar usuários
  Para que eles sejam exibidos corretamente na listagem

====
CRUD
====

Cenário: Criar usuário com dados válidos
  Dado que estou na tela de usuários
  Quando clico no botão adicionar usuário
  E preencho todos os campos obrigatórios com dados válidos
  E seleciono uma empresa válida
  E clico em salvar
  Então o usuário deve ser exibido na listagem

Cenário: Visualizar dados de um usuário cadastrado
  Dado que existe um usuário cadastrado
  E que estou na tela de usuários
  Quando seleciono o usuário na listagem
  Então os dados do usuário devem ser exibidos corretamente

Cenário: Editar dados de um usuário
  Dado que existe um usuário cadastrado
  E que estou na tela de usuários
  Quando edito os dados do usuário
  E salvo as alterações
  Então os dados atualizados devem ser exibidos na listagem

Cenário: Excluir um usuário
  Dado que existe um usuário cadastrado
  E que estou na tela de usuários
  Quando excluo o usuário
  Então o usuário não deve ser exibido na listagem

==========
CAMPO NOME
==========

Cenário: Não permitir cadastro com nome vazio
  Dado que estou na tela de usuários
  Quando clico no botão adicionar usuário
  E deixo o campo nome vazio
  E preencho os demais campos com dados válidos
  E clico em salvar
  Então o sistema deve exibir uma mensagem solicitando o preenchimento do campo nome

  E clico em salvar
  Então o sistema deve impedir o cadastro
